Gestor de Base de Datos

Java

JavaFX

MySQL

Este programa de escritorio facilita la administración de una base de datos permitiendo realizar operaciones de lectura, escritura, actualización y eliminación de datos.

Para conectarse a una base de datos se necesita ingresar la dirección, el usuario, contraseña y el nombre de la base de datos que se desea gestionar.

Repositorio

Al ingresar, se muestra el listado de tablas de la base de datos y al seleccionar una tabla se podrán realizar las consultas de lectura, escritura, etc. Al presionar sobre alguna de las tablas listadas se mostrarán las columnas correspondientes en donde se listarán los datos de las consultas que se realicen.
