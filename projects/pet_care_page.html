Pet Care

Spring Boot

Flutter

MySQL

PetCare es una aplicación web para la administración de una veterinaria. En donde se pueden agendar citas, ver el listado de veterinarios, manejar productos y reportes.

El proyecto consistió en el mantenimiento de la aplicación web existente y el desarrollo de una aplicación móvil para ver el listado de productos con sus características.

En el mantenimiento de la aplicación web se realizó la mejora de la interfaz visual, se agregó un apartado de adquisición de productos por parte de clientes, sección de citas, reportes de citas y la característica responsiva.

Repositorio

Se desarrolló una aplicación móvil para ver el listado de productos de la aplicación Pet Care. Gracias a una API que se creó en la aplicación web se puede acceder con usuario y contraseña para ver todos los productos existentes de la veterinaria.
