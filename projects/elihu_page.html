EliHu

Java

JavaFX

EliHu es un programa que realiza la labor de un intérprete. Con las palabras reservadas aceptadas por EliHu se pueden realizar ciertas tareas que un lenguaje de programación puede realizar, tal como imprimir mensajes, ejecutar condicionales, ejecutar ciclos, crear e invocar funciones y realizar algunas operaciones aritméticas. Al introducir una porción de código, un proceso es ejecutado y pasa por la etapa de análisis léxico, seguido de un análisis sintáctico y termina con un proceso de análisis semántico. Con lo anterior se podrá ejecutar el código escrito y mostrar una salida.
