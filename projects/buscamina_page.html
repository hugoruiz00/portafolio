Este juego de buscaminas desarrollado con HTML, CSS y Javascript cuenta con tres niveles, entre mayor es el nivel, mayor es el tamaño del tablero y mayor el número de bombas. Cuenta con un temporizador para llevar el tiempo de la partida si así se desea. El objetivo es encontrar todas las caritas sin tocar ninguna bomba, para esto se muestra el número de aciertos y la cantidad restante para poder ganar la partida.
